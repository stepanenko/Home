## Some NodeJS projects during IT Academy

### Heroku commands:

Install the Heroku CLI ( download / npm / brew ... )

**Login:**

- `heroku login`

**To clone the repository:**

- `heroku git:clone -a <name_of_app>`

**Deploy your changes:**

- `git add .`
- `git commit -am "made it better"`
- `git push heroku master`
